Cybercrime Law protest initiated by BAND

== Description ==

This WordPress plugin inserts a modal window stating our protest for Cybercrime Law.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the `no-to-cybercrime-law` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= How do I deactivate it? =

Just login to your Wordpress admin, go to Plugins and click Deactivate under No to Cybercrime Law
